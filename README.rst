django-pglock
########################################################################

Documentation
=============

`View the django-pglock docs here
<https://django-pglock.readthedocs.io/>`_.

Installation
============

Install django-pglock with::

    pip3 install django-pglock

After this, add ``pglock`` to the ``INSTALLED_APPS``
setting of your Django project.
